# What is LINQ?

- LINQ stands for Language-Integrated Query.
- LINQ is developed by Microsoft.
- Based on Microsoft documents:
    `LINQ is the name for a set of technologies based on the integration of query capabilities directly into the C# language`


## Different type of data sources

- Collections
- ADO.Net Datasets
- XML Documents
- Entity Framework
- SQL Database

## Some benefits of LINQ

- Compile time safety
- Using same Code for all data sources:
- Increase Code Readability
- IntelliSense Support
